#### Какие функции должны быть в одноранговом платежном приложении?

Хотя большинство одноранговых платежных приложений выглядит одинаково, не каждый, доступный на рынке, одинаково прост и безопасен. Все зависит от особенностей этих приложений похвастаться. Чтобы создать успешное одноранговое приложение, вам необходимо включить селдующие функции:
* Каждый пользователь должен иметь уникальный идентификатор и профиль.
* Когда деньги отправлены или получены, пользователи должны иметь возможность видеть все детали транзакции.
* Должен быть простой чат, чтобы пользователи могли обсуждать транзакции в словах.

* Должна быть определена поддержка для разрешения споров с случае их возникновения.
